# Red-Team-Runner
A red team runner designed to test an LLM's capabilities against threats like prompt injection, jail break, data exfiltration and toxic requests

## Overview
Scenario-driven red-teaming runner for mock GenAI apps.

## Features
- YAML-based attack scenarios
- Deterministic mock chatbot
- Failure detection via keywords
- REST API + CLI + UI
